# Nova Heaven

> Free, premium-quality WordPress security and performance plugins. Built by an independent developer in Tennessee.

**Nova Core** and **Nova Scan** are free forever. Every other Nova Heaven plugin (Hyper Nova, Nova Ascend, Nova Burst, Nova Shield) is paid — the paid plugins fund Nova Core and Nova Scan staying free.

## Plugins

- **[Nova Scan](https://novaheaven.io/en/novascan)** — free WordPress malware scanner, learning detection engine
- **[Hyper Nova](https://novaheaven.io/en/hypernova)** — advanced caching engine with intelligent warming
- **[Nova Core](https://novaheaven.io/en/novacore)** — framework required by all Nova plugins
